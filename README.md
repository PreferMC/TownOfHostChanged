* 随着国内开发环境变化，TOH分支已经不具有开发潜力，并且本人不具有开发MOD的精力。
* 当前国内MOD开发者数量屈指可数，本MOD规范代码初衷没有了实现意义。
* 当前需要的是更好的用户面向性MOD而不是开发者面向性MOD，当前已经有了面向用户舒适性的MOD。
